# vue-cesium-template
一个基于Vue的简易Cesium开发模板

## 项目初始化
```
npm install
```

补全`src/common/js/config.js`中的`Token`信息：
- `CesiumIonDefaultAccessToken`：前往 https://cesium.com/ 中注册账号，并且创建`Token`；
- `TianDiTuToken`：前往 https://uums.tianditu.gov.cn/register 中注册账号，并且创建`Token`。

### 编译
```
npm run serve
```

### 打包
```
npm run build
```
